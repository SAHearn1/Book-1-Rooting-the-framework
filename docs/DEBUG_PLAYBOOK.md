## Common Failure Patterns

| Symptom | First Check | Likely Cause |
|---------|------------|---------------|
| Build fails | Build logs | Missing dependency or config |
| Links broken | href paths | Renamed files or wrong path |
| Images missing | src paths | Wrong path or missing asset |
| Export not generating | Export script | Script config or dependency |
